隐私政策

最后更新：2026年5月

1. 概述

BudgetApp 是一款客户端个人理财工具。我们致力于保护您的隐私。本政策说明了我们存储哪些数据以及如何使用这些数据。

2. 我们存储的数据

BudgetApp 仅将您的预算条目（标题和金额）存储在您浏览器的 localStorage 中。这些数据不会离开您的设备，也不会传输到任何服务器。

3. Cookie 与本地存储

我们使用 localStorage 实现两个功能：

entry_list — 存储您的收入和支出记录，使其在会话之间保持持久。

cookie_accepted — 记录您是否已确认本隐私声明，以避免每次访问都显示提示横幅。

本应用不使用任何追踪 Cookie、广告 Cookie 或第三方 Cookie。

4. 第三方服务

BudgetApp 不集成任何第三方分析、广告或数据收集服务。所有功能均在您的浏览器本地运行。

5. 数据保留与删除

您的数据仅存储在您自己的设备上。您可以随时通过清除浏览器的 localStorage 来删除所有存储的数据（开发者工具 → Application → Local Storage → Clear）。

6. 联系方式

如果您对本政策有任何疑问，请在我们的 GitHub 仓库中提交 Issue。
